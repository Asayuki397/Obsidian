# Definition

A [[Random Variable]] that can only have 1 or 0

### Example
$$
X_{i} = \begin{cases}
1 \text{ for x = 1}\\
0 \text{  otherwise}
\end{cases}
$$
# Facts

let
$$
\begin{align}
P(X=0) = 1-p \\
P(X=1) = p
\end{align}
$$
we get:
$$
EX = p
$$
$$
Var X = E[X^2] - (EX)^2 = p-p^2
$$
