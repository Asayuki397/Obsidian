### Square Matrix
정방행렬, 정사각행렬

_m = n_ => 행렬 ***A*** = \[a<sub>ij</sub>]<sub>n x n</sub>를 n차 Square Matrix 라고 한다.
또한 a<sub>11</sub>, a<sub>22</sub>, ... , a<sub>nn</sub> 을 ***A***의 main diagonal elements 라고 한다.
